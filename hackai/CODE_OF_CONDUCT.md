# 행동 강령

이 저장소는 보안 연구와 지식 공유를 위한 공간입니다.  
모든 참여자는 다음 원칙을 지켜 주세요.

---

## 기본 원칙

### 1. 상호 존중

- 배경, 경력, 관점에 관계없이 모든 참여자를 존중합니다.
- 비판은 아이디어에 집중하고, 사람을 대상으로 하지 않습니다.
- 공격적, 차별적, 모욕적 표현은 사용하지 않습니다.

### 2. 사실 기반 토론

- 주장에는 근거를 제시합니다.
- 확인되지 않은 정보를 사실처럼 제시하지 않습니다.
- 보안 기술 주제는 검증 가능한 자료를 기반으로 논의합니다.

### 3. 책임 있는 보안 연구

- 이 저장소에서 공유되는 지식은 방어와 교육 목적에만 사용합니다.
- 보안 취약점 정보를 비인가 시스템 공격에 활용하지 않습니다.
- 실험은 명시적으로 허가된 환경에서만 수행합니다.
- 발견된 실제 취약점은 책임 있는 공개(responsible disclosure) 절차에 따라 처리합니다.

### 4. 위험한 콘텐츠 제한

- 공격 코드, 익스플로잇, 취약점 재현 절차, 우회 기법을 이 저장소에 게시하지 않습니다.
- 다른 사람이 비인가 시스템에 피해를 줄 수 있도록 돕는 정보를 제공하지 않습니다.
- "교육 목적"을 명분으로 실제 공격에 활용 가능한 코드나 절차를 공유하지 않습니다.

---

## 허용되지 않는 행동

- 참여자에 대한 괴롭힘, 위협, 스토킹
- 개인정보(이름, 소속, 연락처 등) 무단 공개
- 공격 도구, 페이로드, 익스플로잇 코드 게시
- 비인가 시스템 점검 또는 공격 방법 공유
- 허위 사실 또는 허위 연구 결과 유포
- 다른 참여자의 기여를 고의로 방해하거나 훼손

---

## 위반 처리

행동 강령 위반 사항은 저장소 관리자에게 신고할 수 있습니다.  
위반의 심각성에 따라 경고, 기여 제한, 차단 등의 조치를 취할 수 있습니다.

위반 신고: GitHub Issue 또는 저장소 관리자 직접 연락

---

## 범위

이 행동 강령은 다음 공간에 적용됩니다:

- GitHub Issues, Pull Requests, Discussions
- 이 저장소와 연관된 공개 커뮤니케이션 채널
- 이 저장소의 콘텐츠를 발표, 공유, 인용하는 경우

---

이 행동 강령은 [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)을 기반으로 작성되었으며, 이 프로젝트의 보안 연구 특성에 맞게 수정되었습니다.
